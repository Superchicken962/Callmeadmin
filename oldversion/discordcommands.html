Discord Bot Commands

c!rainbowme - Gives you the rainbow role that changes colour when bot is online (This is only beta and may not work very often)

c!unrainbowme - Pretty clear, Removes the rainbow role from you.

c!currency - Shows how many AdminBucks you currently have.

c!currencyother - See how many AdminBucks someone else has when you ping them after the command. Please do not spam this.

c!achievements - View your achievements. There are currently 2 achievements you can unlock.

Currently Disabled. Check bugs and glitches below for reasons.
c!rtd - Roll The Dice!

c!play - Plays a video on youtube in a voice chat, c!play

c!stop - Stops the current music playing and leaves voice chat

c!pause - Pauses the current music playing (use c!resume to unpause)

c!resume - Resumes the music that was being played (use c!pause to pause)

c!website - View the official CallMeAdmin website! (You're on it right now!)

c!serverstats - See stats of any online tf2 server! (Use: c!serverstats [ip] [port] )

(Coming soon) c!game - See someone's current status. (Use: c!game [user id] )

Admin Commands

WARNING! This command is unstable and may not run well, use it at your own risk.
c!rainbow_role_on/off - Turn on turn off rainbowmode on a certain role. Use: c!rainbow_role_on [role]

c!mute - Adds a mute role to the user you specify with their user id. (make sure you have added a 'Muted' role with working perms before you use this command.

c!kick - Kicks a member from the server. (c!kick ) then sends a dm to the kicked member with the reason of their kick

Caution. This command has some issues, please read the bugs below.
c!ban - Bans a user from the server for a specified amount of time. (c!ban )


Bugs and Glitches

Solution categories
* - Solution is known, the command is just waiting to be fixed.
X - No solutions known at the moment.
F - Command is in the process of being fixed.
U - Issue is unknown.

X c!ban - When time of ban runs out, the bot can't unban the member and if you want to use the command, you can manually unban the member when the time runs out.

* c!rtd - Cooldown timer glitches and doesn't actually turn off when 3 seconds has passed.
